# 🐱 Neko_Han 猫娘养成机器人

喵～欢迎来到 **Neko_Han** 的世界！这是一个基于 [AstrBot](https://github.com/Soulter/AstrBot) 框架的多功能猫娘养成机器人，由 **Stalyx（创造君）** 开发。你可以创建属于自己的猫娘，通过签到、劳动、决斗、婚恋等方式积累猫条，享受养猫的乐趣喵～

## ✨ 特性

- 🐈 **猫娘养成**：创建与管理你的专属猫娘，自定义名字
- 💰 **经济系统**：猫条作为通用货币，签到、劳动、决斗均可赚取
- ⚡ **体力系统**：每日重置体力，用于接取劳务任务
- 🏗️ **劳动市场**：每小时刷新多种劳务任务，消耗体力可获得猫条
- ⚔️ **决斗系统**：向其他猫娘发起赌注猜数字的决斗，一决高下
- 💕 **婚恋系统**：求婚、接受求婚、离婚，构建猫娘之间的甜蜜关系
- 📊 **信息聚合**：仪表盘一站式查看猫条、体力、婚姻、劳务和决斗邀请

## 🛠️ 安装与运行

### 环境要求
- Python 3.8+
- MongoDB（本地或远程实例）
- AstrBot 框架及必要插件

### 步骤
1. 克隆项目或下载插件代码，放入 AstrBot 的插件目录。
2. 安装依赖：
   ```bash
   pip install motor
   ```
3. 启动 MongoDB 服务（默认连接 `mongodb://localhost:27017`，可在代码中修改）。
4. 启动 AstrBot 主程序，插件会自动注册。

## 📚 指令列表

### 👋 基础互动
| 指令 | 说明 |
|------|------|
| `/喵` | 和 Neko_Han 打个招呼 |
| `/你是猫娘么` | 问问它是不是猫娘 |

### 🐈 猫娘养成
| 指令 | 说明 |
|------|------|
| `/创建猫娘 <名字>` | 创建你的猫娘（名字不能有空格） |
| `/修改名字 <新名字>` | 修改猫娘的名字 |
| `/仪表盘` | 查看猫条、体力、婚姻、劳务、决斗邀请等信息 |

### 💰 经济系统
| 指令 | 说明 |
|------|------|
| `/签到` | 每日签到，随机获得 10~50 根猫条 |
| `/转账 <猫娘名字> <数量>` | 向另一只猫娘转账猫条 |

### 🏗️ 劳动系统
| 指令 | 说明 |
|------|------|
| `/猫力资源市场` | 查看可接劳务任务列表（每小时刷新） |
| `/接取劳务 <序号>` | 接取任务，消耗体力，完成后得猫条 |
| `/我的劳务` | 查看进行中的任务及剩余时间 |
| `/取消劳务 <序号>` | 取消任务（体力不返还，任务退回大厅） |

### 💕 婚恋系统
| 指令 | 说明 |
|------|------|
| `/求婚 <猫娘名字>` | 让你的猫娘向其他猫娘求婚 |
| `/接受求婚 <猫娘名字>` | 接受对方的求婚 |
| `/离婚` | 与当前配偶离婚 |

### ⚔️ 决斗系统
| 指令 | 说明 |
|------|------|
| `/决斗 <猫娘名字> <赌注> <你的猜测>` | 向其他猫娘发起猜数字决斗（0-100） |
| `/接受决斗 <发起者名字> <你的猜测>` | 应战决斗邀请 |
| `/拒绝决斗 <发起者名字>` | 拒绝决斗邀请 |
| `/决斗列表` | 查看所有待处理的决斗邀请 |

### ❓ 帮助
| 指令 | 说明 |
|------|------|
| `/你能做什么` | 显示完整的帮助信息 |

## 📡 数据存储

所有数据存储在 MongoDB 的 `neko-han` 数据库中，包含以下集合：
- `nekoes`：猫娘信息（用户ID、名字、猫条、体力、婚姻状态等）
- `labor_hall`：劳务任务（状态、消耗、奖励、时间等）
- `duel_invites`：决斗邀请（发起者、目标、赌注、猜测、状态）

体力每日自动重置，劳动大厅每小时刷新，过期任务自动完成并发放奖励。决斗邀请在完成后或拒绝后自动清除。
